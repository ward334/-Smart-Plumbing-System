# SPS - Smart Plumbing System

## نظام السباكة الذكي للهاكاثون

### وصف المشروع
SPS هو نظام سباكة ذكي متكامل مع مستشعرات IoT والذكاء الاصطناعي. يهدف إلى اكتشاف والتنبؤ بتسريبات وانسدادات أنابيب المياه في المبنى قبل حدوثها، وتحديد موقعها بدقة في الشبكة.
رابط الموقع (Demo):https://spsplumbin-xzjwqa9x.manus.space
### الميزات الرئيسية
- **مستشعرات IoT**: مراقبة تدفق المياه والضغط ودرجة الحرارة في الوقت الفعلي
- **كشف الذكاء الاصطناعي**: خوارزميات التعلم الآلي لتحليل الأنماط والتنبؤ بالتسريبات
- **تنبيهات فورية**: إشعارات فورية عند اكتشاف أي شذوذ مع معلومات الموقع الدقيقة
- **خرائط تفاعلية**: مخططات المباني التفاعلية تظهر مواقع المستشعرات والتسريبات

### كيفية الاستخدام

#### الطريقة 1: فتح مباشر
1. افتح ملف `index.html` في أي متصفح ويب
2. اضغط على زر "Login"
3. استخدم بيانات الدخول التجريبية: `admin` / `demo123`
4. استكشف صفحات التطبيق

#### الطريقة 2: خادم محلي (مستحسن)
```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx serve .

# ثم افتح http://localhost:8000
```

### الصفحات

#### 1. صفحة الترحيب (index.html)
- عرض شعار وعنوان المشروع
- زر تسجيل الدخول

#### 2. صفحة تسجيل الدخول (login.html)
- حقول اسم المستخدم وكلمة المرور
- بيانات تجريبية: admin / demo123

#### 3. الصفحة الرئيسية (home.html)
- وصف المشروع والميزات
- إحصائيات سريعة للنظام
- زر الانتقال للمخطط

#### 4. صفحة المخطط (blueprint.html)
- عرض مخطط المبنى مع نقاط المستشعرات
- **رفع مخطط**: يمكن رفع صورة مخطط مخصصة
- **محاكاة التسريب**: اختيار مستشعر لمحاكاة تسريب
- **إعادة تعيين النظام**: إعادة جميع المستشعرات للحالة الطبيعية
- تلميحات عند التمرير على المستشعرات

#### 5. صفحة التنبيهات (alerts.html)
- قائمة بجميع التنبيهات المسجلة
- النقر على تنبيه ينتقل للمخطط ويبرز المستشعر

#### 6. لوحة التحكم (dashboard.html)
- إحصائيات النظام (عدد المستشعرات، التسريبات، صحة النظام)
- رسم بياني لتكرار التسريبات
- تنبؤات مخاطر الذكاء الاصطناعي
- معلومات النظام وقدرات AI

### التخزين المحلي
يستخدم التطبيق `localStorage` لحفظ:
- بيانات المستشعرات وحالتها
- سجل التنبيهات
- صورة المخطط المرفوعة

### التخصيص

#### تغيير الألوان
عدّل المتغيرات في بداية ملف `css/styles.css`:
```css
:root {
  --primary: #0ea5e9;        /* اللون الأساسي */
  --destructive: #ef4444;    /* لون التنبيهات */
  --warning: #f59e0b;        /* لون التحذيرات */
  --success: #10b981;        /* لون النجاح */
}
```

#### إضافة مستشعرات
عدّل مصفوفة `sensors` في ملف `js/app.js`:
```javascript
const SPSData = {
  sensors: [
    { id: 1, name: "Sensor #1", location: "Kitchen", x: 30, y: 25, status: "active" },
    // أضف مستشعرات جديدة هنا
  ],
  // ...
};
```

### التقنيات المستخدمة
- **HTML5**: هيكل الصفحات
- **CSS3**: التصميم والرسوم المتحركة
- **Vanilla JavaScript**: المنطق والتفاعلية
- **LocalStorage**: تخزين البيانات محلياً
- **Google Fonts**: خطوط Outfit و DM Sans

### متطلبات التشغيل
- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- لا يتطلب أي خادم أو قاعدة بيانات
- يعمل بشكل مستقل تماماً
